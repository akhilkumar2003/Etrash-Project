# ♻️ ETrash+ Digital Platform

A smart digital solution designed to manage and streamline electronic waste (E-Waste) collection and awareness. This platform helps users contribute to a cleaner environment by promoting responsible disposal of electronic items.

---

## 🌟 Features

- 🧾 User-friendly interface for managing e-waste
- 📦 Track and manage electronic waste records
- 🔍 Easy navigation and data handling
- 🌐 Web-based application accessible via browser
- 💾 Secure data storage using MySQL

---

## 🛠️ Tech Stack

### 💻 Backend
- Java (Core Java)
- Servlets & JSP

### ⚙️ Framework & Tools
- Spring Tool Suite (STS)
- JDBC (Java Database Connectivity)

### 🗄️ Database
- MySQL
- Managed using MySQL Workbench

### 🌐 Server
- Apache Tomcat Server

---

## 🏗️ Development Environment

This project was developed using:

- **IDE:** Spring Tool Suite (STS)
- **Server:** Apache Tomcat
- **Database Tool:** MySQL Workbench
- **Language:** Java (Servlets, JSP)
- **Connectivity:** JDBC

---

## 🔗 How It Works

1. User interacts with the web interface
2. Requests are handled using Servlets
3. Data is processed and stored using JDBC
4. MySQL database stores all records
5. Apache Tomcat server runs the application
